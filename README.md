# དྲ་ཚིགས་འདིའི་ནང་ཡིག་ཆ་རྩོམ་སྒྲིག་བྱེད་སྟངས་ངོ་སྤྲོད།

དྲ་ཚིགས་འདིའི་ནང་གི་ཡིག་ཆའི་ཡོང་ཁུངས་ནང་བསྟན་དཔེ་ཚོགས་ལྟེ་གནས་སམ་BDRC གྱི་དྲ་ཐོག་དཔེ་མཛོད་ཡིན། དཔེར་ན། འདིའི་དཀར་ཆག་ནང་བཀོད་པའི་ཨང་གྲངས་འདི་ W3CN15333 བརྩམས་ཆོས་ཨང་རེད། ཨིན་སྐད་ལ།(Work Number) ངེད་ཚོགས་པའི་དྲ་ཐོག་དཔེ་མཛོད་ནང་གི་དཔེ་ཆ་རེ་ལ་འདི་ལྟ་བུའི་ཨང་གྲངས་རེ་སྤྲད་ཡོད། ཁྱེད་ལ་མཁོ་བའི་དཔེ་ཆའི་མཚན་བྱང་སྤྱད་ནས་ངེད་ཚོགས་པའི་དྲ་ཚིགས་ tbrc.org ནང་
