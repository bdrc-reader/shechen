ཨང་གྲངས་འདི་ནི་W1ERI0009 དགོན་པ་དང་ཡང་ན་དཔེ་མཛོད་སོགས་སོ་སོ་མཚོན་བྱེད་ཀྱི་ཨང་ཡིན། དེའི་རྗེས་ཀྱི་ཨང་001 འདི་སྡེ་ཚན་རེ་རེ་མཚོན་བྱེད་ཡིན་པས། སྡེ་ཚན་དང་པོ་ལ་W1ERI0009001 དང་གཉིས་པ་ལ་W1ERI0009002 ལྟ་བུ།  གཞན་ལ་རིགས་འགྲེ་ནས་ཤེས་པར་བྱའོ།། གོང་གི་པར་ངོས་སུ་གསལ་བ་ལྟར་ཐིག་ཕྲེང་གཅིག་གི་སྟེང་བརྗོད་གཞི་གཅིག་པའི་ཚིག་དུམ་མམ་ཚིག་གཅིག་མ་གཏོགས་སྔ་ཕྱི་བྱས་ནས་བརྗོད་གཞི་མ་འདྲ་བའི་རིགས་གཉིས་འཇོག་མི་རུང་། མདོར་ན། ཐིག་ཕྲེང་གཅིག་གི་སྟེང་བརྗོད་གཞི་གཅིག་རེ་འཇོག་དགོས།

## ག ཤོག་གྲངས་དང་པོ་ཏིའི་ཨང་།

<br>

འཛིན་གྲྭ་སོ་སོའི་བར་དང་ཡང་ན་སློབ་དུས་སོ་སོའི་བར་ལ་གཞུང་ལུགས་གཅིག་གི་ཁྲིད་མཚམས་དང་ཡང་ན་རྩོད་མཚམས་ཀྱི་ཤོག་གྲངས་དང་པོ་ཏིའི་གྲངས་བཅས་བཞག་ཡོད། ཤོག་གྲངས་དང་པོ་དང་གཉིས་པ་པོད་ཨང་འཇོག་དགོས། དཔེར་ན།  (23,100,1,1) ཤོག་གྲངས་༢༣ ནས་༡༠༠ བར་དང་པོད་དང་པོའི་ནང་ཡོད་པ་མཚོན། འཇོག་སྟངས་བརྩམས་ཆོས་ཨང་དེ་དང་ཤོག་གྲངས་དང་པོད་ཨང་གཉིས་ཀྱི་བར་སྟོང་མེད་པར་སྦྱར་ནས་འཇོག་དགོས། དཔེར་ན། W7494(23,100,1,1) གཤམ་གྱི་དཔེ་རིས་ལ་གཟིགས།
